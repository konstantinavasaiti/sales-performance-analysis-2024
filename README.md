# Sales Performance Analysis 2024

## Project Overview

This project simulates a real-world Financial Data Analyst assignment.

The objective is to analyze the company's sales performance using Python, SQL and Power BI in order to provide actionable business insights to management.

Project Status

🟢 Data Understanding


## “Phase 2 – Exploratory Data Analysis”

Phase 2 – Exploratory Data Analysis is the first major step toward understanding the company’s commercial performance. The goal is to answer the CFO’s initial business questions through reproducible, clean, and well‑documented analysis.

This phase includes a structured Jupyter Notebook with all calculations generated programmatically (no manual copy‑paste of results).

🔍 Executive KPIs
Key performance indicators that summarize overall sales performance:

Total Net Sales – total revenue after discounts

Total Orders – number of unique orders (Distinct InvoiceID)

Total Customers – number of unique customers

Average Order Value (AOV) – average revenue per order

Total Quantity Sold – total units sold

Total Profit – total profit generated

Profit Margin % – profitability ratio

📈 Sales Trend Analysis
Monthly performance insights:

Sales by Month

Profit by Month

Trend visualizations for quick interpretation by the CFO

🛒 Product Analysis
Evaluation of product and category performance:

Top 10 Products (by Net Sales)

Top Categories

Identification of high‑performing and underperforming products

👥 Customer Analysis
Understanding customer value and behavior:

Top Customers

Revenue per Customer

Detection of high‑value customers

🌍 Regional Analysis
Geographical performance overview:

Revenue by Region

Identification of strong and weak markets

🔄 Reproducibility & Clean Code
The analysis is fully reproducible and follows best practices:

Clear notebook structure with explanatory markdown

Clean, commented Python code

Use of relative paths for data loading

All KPIs and charts generated directly from code

No manual insertion of results

Separation of raw data, processed data, and notebooks
